# OpenTune

A free, open-source Android music player powered by YouTube Music, built with Jetpack Compose.

[![Build](https://github.com/tigerskin-hupee/opentune/actions/workflows/build.yml/badge.svg)](https://github.com/tigerskin-hupee/opentune/actions/workflows/build.yml)
[![License: GPL v3](https://img.shields.io/badge/License-GPLv3-blue.svg)](https://www.gnu.org/licenses/gpl-3.0)

## Features

- Stream music via yt-dlp (no login required)
- Offline download for local playback
- Synced lyrics display
- Android Auto support
- Backup / restore (compatible with OuterTune backups)
- Material You design

## Building

```bash
./gradlew assembleDebug
./gradlew assembleRelease
```
